Community Metrics · Q2 2026

LanceDB community
metrics — Q2 2026.

Package download trends for the Python, TypeScript, and Rust SDKs through May 31, 2026. What moved, and what we can infer about why.

Prashanth Rao DEVREL · LANCEDB
02/08
Summary

TL;DR

  • Python (lancedb) is the largest SDK and still growing: +19% MoM, all-time high in May (8.11M).
  • TypeScript (@lancedb/lancedb) dropped 37% MoM to 2.77M, off an anomalous April peak (4.40M). Still ~2× March.
  • Rust (lancedb) is small but growing fast: +21% MoM, all-time high (98.5K).
  • Lance format (pylance, lance crate) is flat to slightly up.

All figures are package download counts (PyPI / npm / crates.io). Causes discussed later are hypotheses — downloads carry no referrer, so none of this proves causation.

TypeScript SDK

TypeScript: April spike, May drop

@lancedb/lancedb · monthly npm downloads · Jan–May 2026
  • Apr 2026: 4.40M — ~3× the prior month.
  • May 2026: 2.77M — −37% MoM, still ~2× March.
  • Pattern is a spike normalizing, not a declining baseline.
Observed by Prashanth

Prashanth attributes the April spike to OpenClaw buzz around the TS memory plugin. Plausible, but not verifiable from download data.

Investigation

Did the Lance-DuckDB posts move downloads?

Avg downloads/day before vs after May 20 (post date). The increase shows up on the Lance format package, not the LanceDB SDK.

bars scaled per package · avg downloads/day
  • pylance: +25.6% after May 20, sustained ~2 weeks.
  • LanceDB Python SDK: −9.8% over the same window.
  • pylance is the package a DuckDB + Lance workflow installs.
Blog timing from Prashanth

Correlation only. Downloads carry no referrer, so this can't confirm the posts caused the increase.

07/08
Next steps

Recommendations

  • TypeScript: capture trial installs from the April spike before they churn — docs beyond agent-memory, plus a production/deploy guide.
  • Lance-DuckDB: the pylance increase correlates with the posts. Ship more format-interop content.
  • June: check whether @lancedb/lancedb settles above its pre-April baseline or keeps dropping.
  • Attribution: pair download data with docs/referral analytics to test these hypotheses.
